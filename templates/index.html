Insurance API

A REST API for AI voice bot insurance services

Getting Started

This API provides endpoints for an AI voice bot to handle insurance-related queries including payment assistance, provider lookup, and benefits information.

All endpoints return data in standard JSON structure with proper HTTP status codes and error handling.

View API Documentation

Core Functionality

Premium Payment

Authenticate users and provide carrier payment portal information

Provider Lookup

Find in-network doctors based on location and specialty

Plan Benefits

Retrieve coverage details, deductibles, and copay information

Available Endpoints

  • GET /api/health - Health check endpoint
  • POST /api/payments/authenticate - Authenticate for payment
  • GET /api/payments/portal - Get payment portal info
  • GET /api/providers - Find in-network providers
  • GET /api/providers/check-network - Check if provider is in-network
  • POST /api/benefits/authenticate - Authenticate for benefits
  • GET /api/benefits/plan/{plan_id} - Get plan benefits
  • GET /api/benefits/coverage - Check service coverage
